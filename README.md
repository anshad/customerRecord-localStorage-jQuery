customerRecord-localStorage-jQuery
==================================

Customer Record App using HTML5 localStorage,jQuery and Twitter Bootstrap

Features
==========

Add Customer to localStorage,	
List all localStorage data in grid,		
Edit individual customer,	
Delete a customer,	
Clear all records.
